# WEATHER VISUALIZER

사물 인터넷 팀 프로젝트 결과물 코드입니다. (2022-06)


## 🛠 사용한 재료
| 물품 | 내용 |
| -- | -- |
| NodeMCU 1.0 | WIFI를 이용하여 현재 날씨와 시간을 불러오기 위해 사용 |
| 10파이 RGB LED | 구름 색을 표현하기 위해 사용 |
| 3파이 LED (yellow, blue) | 지구 야경, 비를 표현하기 위해 사용 |
| 5파이 LED (yellow, red) | 달과 태양을 표현하기 위해 사용 |
